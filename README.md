## Features
- ✅ Parsing with quotes and special characters
- ✅ Built-in `cd` and `path`
- ✅ Background execution (`&`)
- ✅ Pipeline (`|`) with unlimited commands
- ✅ Redirections: `>`, `>>`, `<`, `2>`, `2>&1`
- ✅ Logical operators: `;`, `&&`, `||`
- ✅ `setpath`, `addpath`, `resetpath`

make
